# Art Rules

- All art lives in `res://Art/`. Use **forward slashes** and `res://` resource paths.
- Filenames are keys. The default **art id** is the lowercase filename without extension.
  Examples:
  - `res://Art/art_sap_shot.png`  → id: `art_sap_shot`
  - `res://Art/bark_shield.png`    → id: `bark_shield`
  - If a card/enemy/sigil wants a shorter id like `sap_shot`, code should try these in order:
  1) exact id,
  2) `art_` + id,
  3) `sigil_` + id,
  4) fallback: `res://Art/card_back.png`.
- Do **not** rename files in code; change the id or add a mapping entry instead.
- On startup, the game builds an in-memory map of every `*.png` in `res://Art/`.
